## Quick Start

Use the navigation to browse the documentation by topic. The content is organized as a production-ready knowledge base for stakeholders, engineers, and delivery teams.

## Build Locally

Run the following command from the repository root:

```bash
source .venv/bin/activate
mkdocs serve
```

Then open http://127.0.0.1:8000/edusync-docs/ in your browser.
