# Diagrams

This folder is the central repository for all EduSync documentation diagrams.

It contains source files, exported assets, and supporting documentation for architecture, business, database, API, security, deployment, and wireframe-related diagrams.

## Purpose

Use this directory to organize diagrams according to their domain and intended audience. Diagrams should remain version controlled, maintainable, and linked from the relevant documentation pages.

## Structure

- architecture/: system and component architecture diagrams
- database/: entity relationship and data model diagrams
- business/: business workflows, process flows, and domain interaction diagrams
- api/: API interaction and integration diagrams
- security/: authentication, authorization, and security architecture diagrams
- deployment/: runtime, infrastructure, and deployment topology diagrams
- wireframes/: UI flows, wireframes, and experience diagrams
- exports/: exported image assets for documentation and presentations
