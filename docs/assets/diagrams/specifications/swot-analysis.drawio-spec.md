# draw.io Design Specification — SWOT Analysis

- File name: swot-analysis.drawio
- Source context: docs/01-Vision/vision.md

## Purpose
Present the strategic SWOT analysis for the EduSync platform in a professional 2x2 matrix grid. This format allows executive stakeholders to quickly compare internal factors (Strengths/Weaknesses) against external factors (Opportunities/Threats).

## Components
- **Core Pivot**
  - EduSync SWOT (Central label)
- **Quadrant 1: Strengths (Internal, Positive)**
  - Modern SaaS Architecture
  - AI-Powered Direction
  - Unified Platform Scope
  - Security-First Multi-Tenancy
- **Quadrant 2: Weaknesses (Internal, Negative)**
  - New Brand
  - Implementation Complexity
  - Large Module Scope
- **Quadrant 3: Opportunities (External, Positive)**
  - Digital Payments
  - Parent Mobile Expectations
  - School Group Expansion
  - AI Productivity
- **Quadrant 4: Threats (External, Negative)**
  - Established ERP Vendors
  - Migration Resistance
  - Security and Privacy Risk
  - Integration Policy Changes

## Layout
- Use a landscape canvas with a 2x2 grid layout (four large quadrant boxes).
- Place `Strengths` in the top-left and `Weaknesses` in the top-right.
- Place `Opportunities` in the bottom-left and `Threats` in the bottom-right.
- (Optional) Place a small circular or diamond-shaped badge containing `SWOT` at the center intersection of the grid lines to anchor the diagram.
- Each quadrant should have a clear, bold header with an icon indicator or capital letter (S, W, O, T) and bulleted/stacked rectangular list cards representing the individual items.

## Connections
- Since this is a static matrix model rather than a process flow, there are no connecting arrows between the quadrant items.
- Use clean, minimal visual borders to separate the four quadrants.

## Color Recommendations
- **Strengths (S)**: Pale Green (#F0FDF4) with Emerald Green headers (#16A34A)
- **Weaknesses (W)**: Pale Yellow/Amber (#FFFBEB) with Amber/Orange headers (#D97706)
- **Opportunities (O)**: Pale Blue (#EFF6FF) with Royal Blue headers (#2563EB)
- **Threats (T)**: Pale Red (#FEF2F2) with Crimson/Red headers (#DC2626)
- **Central Badge**: Neutral Slate Gray (#475569) with white text
- **Background**: Soft Gray/Off-White (#F8FAFC)

## Swimlanes
- Swimlanes are represented by the 2x2 grid lines:
  - **Horizontal Divider**: Separates Internal factors (top) from External factors (bottom).
  - **Vertical Divider**: Separates Positive factors (left) from Negative factors (right).
- Label these divider borders as "Internal Factors" (Top), "External Factors" (Bottom), "Helpful / Positive" (Left), and "Harmful / Negative" (Right).
